Powerful API for Easy Integration

Powerful API for developers.

API Overview

Our REST API allows you to integrate AINarrate's text-to-speech capabilities directly into your applications, websites, or workflows. Get programmatic access to generate audio, manage your account, and more. All responses are in JSON format.

// Example API Request (conceptual)
POST /v1/narrate
Authorization: Bearer YOUR_API_KEY
{
  "text": "Welcome to our podcast. Today we'll discuss...",
  "voice": "professional_female",
  "speed": 1.0,
  "format": "mp3"
}

Authentication

Authenticate your API requests by including your API key in the Authorization header. Obtain your API key from your AINarrate dashboard.

Authorization: Bearer YOUR_API_KEY

Key Endpoints

  • POST /v1/narrate: Generate audio narration from text with voice options.
  • GET /v1/audio/{id}: Retrieve status or download link for generated audio.
  • GET /v1/account/usage: Check your current API usage and limits.

Full API documentation will be available upon signing up for an API plan.
