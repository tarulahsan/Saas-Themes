API Overview

Our REST API allows you to integrate SynthCode's image generation capabilities directly into your applications, websites, or workflows. Get programmatic access to generate images, manage your account, and more. All responses are in JSON format.

// Example API Request (conceptual)
POST /v1/generate
Authorization: Bearer YOUR_API_KEY
{
  "prompt": "A futuristic cityscape at sunset, cyberpunk style",
  "style": "photorealistic",
  "aspect_ratio": "16:9"
}

Authentication

Authenticate your API requests by including your API key in the Authorization header. Obtain your API key from your SynthCode dashboard.

Authorization: Bearer YOUR_API_KEY

Key Endpoints

  • POST /v1/generate: Generate a new image based on a prompt and parameters.
  • GET /v1/images/{id}: Retrieve status or details of a generated image.
  • GET /v1/account/usage: Check your current API usage and limits.

Full API documentation will be available upon signing up for an API plan.
